# ☕ Cafe Cart CLI Application (C++)

A simple command-line based cafe ordering system built using C++.
This project demonstrates fundamental programming concepts such as data structures, input handling, and basic system design.

---

## 🚀 Features

* Display cafe menu with prices
* Add items to cart
* Calculate total price
* Input validation and error handling
* Case-insensitive user input
* Dynamic cart using `std::vector`
* Menu storage using `std::unordered_map`

---

## 🧠 Concepts Used

* C++ STL (`vector`, `unordered_map`)
* Input handling (`std::getline`, validation)
* String manipulation
* Looping and control flow
* Basic memory awareness

---

## 🛠️ How to Run

```bash
g++ main.cpp -o cafe
./cafe
```
